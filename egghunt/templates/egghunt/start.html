How it works

The eEaster Egg Hunt works like any other Easter egg hunt. You start with a clue. The clue leads you to a location. At the location, you'll find another clue.

Using the map

To search for a clue at a location, you drop a pin on the map at that location and click "Search Here".

Resuming

You can pause and resume the hunt at any time. Just remember the last location that you visited. When you want to resume, return to that location using the map.

Solving the clues

The clues can be difficult and you're not expected to "just know it". Using Google is allowed. Using Google search-by-image is not allowed. Anything you think is probably cheating, is cheating.

For every clue, your pin needs to be within a certain radius of the true location to be accepted. If you think you have the right answer but it is not accepted, try to get closer to the precise location.
